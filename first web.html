1.คีบอร์ด (Keyboard)

คีย์บอร์ด (Keyboards) คือ อุปกรณ์ที่ใช้นำข้อมูลจากผู้ใช้เข้าสู่เครื่องคอมพิวเตอร์หรือเครื่องมืออิเล็กทรอนิกส์ มีลักษณะเป็นแป้นสี่เหลี่ยม ด้านในมีปุ่มกดเป็นทั้งตัวอักษร ตัวเลข หรือสัญลักษณ์ ซึ่งสามารถป้อนข้อมูลได้หลายภาษา ขึ้นอยู่กับการติดตั้งและเพิ่มภาษานั้นๆ ในระบบคอมพิวเตอร์ ปัจจุบันมีการพัฒนาและใส่ฟังก์ชั่นต่างๆ เพื่อตอบสนองการใช้งานของผู้ใช้ ทำให้คีย์บอร์ดมีหลากหลายรูปแบบ เช่น Wireless Keyboard การเชื่อมต่อแบบไร้สายแทนการใช้สาย USB ใช้งานได้ในระยะที่ไกลขึ้น หรือ Keyboard with Trackball การนำลูกบอลชี้ตำแหน่งไว้บนคีย์บอร์ด เหมาะกับงานที่ต้องการวางจุด หรือรูปภาพ เช่น โปรแกรมเขียนแบบ (CAD) เป็นต้น


2.เมาส์ (Mouse)

หน้าที่ของเมาส์ คือ การควบคุมตำแหน่งของเคอร์เซอร์ (ลูกศร) บนหน้าจอคอมพิวเตอร์ เพื่อให้ผู้ใช้สามารถเลื่อนเคอร์เซอร์ไปยังตำแหน่งต่าง ๆ ตามที่ต้องการได้ โดยเมาส์มีฟังก์ชันหลัก ๆ ดังต่อไปนี้ เคลื่อนที่ (Move) ลากตัวเมาส์เพื่อขยับตำแหน่งของเคอร์เซอร์ที่ปรากฏบนหน้าจอ คลิก (Click) หรือคลิกซ้าย เพื่อเลือกคำสั่งหรือวัตถุบนหน้าจอที่ตำแหน่งของเคอร์เซอร์ ดับเบิลคลิก (Double Click) คลิกซ้ายติดกันสองครั้งบนไอคอนเพื่อเปิดขึ้นมาใช้งาน คลิกแล้วลาก (Drag) ลากคลุมขอบเขตวัตถุที่ต้องการเลือก หรือย้ายตำแหน่งของวัตถุ คลิกขวา (Right Click) เปิดเมนูคำสั่งต่าง ๆ ที่เกี่ยวกับวัตถุที่เลือก ปุ่มล้อเลื่อน (Scroll wheel) เลื่อนหน้าจอขึ้น-ลง สามารถกดเพื่อเปิดโหมดเลื่อนจอได้

3.เคส (Case)

เคสหมายถึง กล่องที่เก็บอุปกรณ์ของคอมพิวเตอร์ข้างในไม่ได้หมายถึง คอมทั้งเครื่องนั่นเอง โดยหน้าที่หลักๆของเคสจะเป็นการจัดเก็บรวบอุปกรณ์ของคอมพิวเตอร์ให้อยู่ข้างในนั้นไม่ว่าจะเป็น Mainboard Ram VGA Powersupply ไปจนถึงอื่นๆนั่นเอง และ อีกหน้าที่หลักๆ คือ ป้องกันอันตรายที่อาจเกิดจากทั้งตัวอุปกรณ์เอง ไม่ว่าจะเป็น ไฟช็อต ไฟดูด ไฟฟ้าลัดวงจร ไปจนถึง ป้องกันอันตรายจากข้างนอกมาใส่อุปกรณ์เช่นของตกใส่เป็นต้น


4.CPU

CPU คือหน่วยประมวลผลกลางของคอมพิวเตอร์เปรียบเหมือนสมองที่คอยทำหน้าที่ในการตัดสินใจหรือคำนวณข้อมูลคำสั่งที่ได้รับมา เป็นหัวใจหลักในการประมวลผล CPU นั้นย่อมาจาก Central Processing Unit ทำหน้าที่ประมวลผลข้อมูลเชิงตรรกะ โดยมีกระบวนการพื้นฐานคือ 1.อ่านชุดคำสั่ง 2.ตีความชุดคำสั่ง 3.ประมวลผลชุดคำสั่ง 4.อ่านข้อมูลจากหน่วยความจำ 5.เขียนข้อมูล/ส่งผลการประมวล ปัจจุบัน CPU อยู่ในรูปแบบของ Socket มีรูปลักษณะเป็นแผ่นสี่เหลี่ยมจัตุรัสทำด้วยเซรามิก ด้านบนกรอบของซีพียูจะบอกรายละเอียดสเปคของซีพียูตัวนั้นๆเมื่อประกอบกับเมนบอร์ดจะดูเหมือนอยู่ในตลับและต้องถูกครอบทับอีกทีด้วยพัดลมหรือชุดน้ำระบายความร้อน


5.การ์ดจอ (GPU)

หน่วยประมวลผลกราฟิก (GPU) เป็นวงจรอิเล็กทรอนิกส์ที่สามารถทำการคำนวณทางคณิตศาสตร์ด้วยความเร็วสูง งานประมวลผลต่างๆ เช่น การเรนเดอร์กราฟิก แมชชีนเลิร์นนิง (ML) และการตัดต่อวิดีโอนั้นต้องนำการดำเนินการทางคณิตศาสตร์ที่คล้ายกันในชุดข้อมูลขนาดใหญ่มาใช้ การออกแบบของ GPU ช่วยให้สามารถดำเนินการเดียวกันกับค่าข้อมูลหลายชุดในแบบคู่ขนานได้ ซึ่งจะเพิ่มประสิทธิภาพในการประมวลผลสำหรับงานต่างๆ มากมายที่เน้นการประมวลผล


6.RAM

RAM เป็นคำย่อของคอมพิวเตอร์ทั่วไปที่ย่อมาจาก Random-Access Memory (หน่วยความจำการเข้าถึงแบบสุ่ม) บางครั้งเรียกว่าหน่วยความจำพีซีหรือสั้น ๆ ว่าหน่วยความจำ โดยหลักแล้ว RAM คือหน่วยความจำระยะสั้นของคอมพิวเตอร์หรือแล็ปท็อปของคุณ เป็นที่จัดเก็บข้อมูลที่โปรเซสเซอร์คอมพิวเตอร์ของคุณต้องมีเพื่อเรียกใช้แอปพลิเคชันและเปิดไฟล์ ภายในคอมพิวเตอร์ของคุณ โดยทั่วไปแล้ว RAM อยู่ในรูปของแผงวงจรแบนสี่เหลี่ยมที่มีชิปหน่วยความจำติดอยู่ หรือที่เรียกว่าโมดูลหน่วยความจำ โดยทั่วไปแล้ว คอมพิวเตอร์มาพร้อมกับโมดูล RAM อย่างน้อยสองโมดูลพร้อมที่ว่างสำหรับเพิ่ม หากจำเป็น โมดูล RAM เหล่านี้เป็นส่วนประกอบสำคัญที่ทำงานร่วมกับหน่วยประมวลผลกลาง (CPU) ของคอมพิวเตอร์ของคุณ และต้องทำงานอย่างเหมาะสมเพื่อให้คุณได้รับประสบการณ์ที่ดี


7.Mainboard

เมนบอร์ด คือ ชิ้นส่วนที่สำคัญอีกชิ้นของคอมพิวเตอร์และโน๊ตบุ๊ค เป็นแผงวงจรกลางเพื่อควบคุมการทำงานของชิ้นส่วนต่างๆ ลักษณะของมันจะเป็นแผ่นสี่เหลี่ยมคล้ายกระดาน มีพอร์ตและช่องต่างๆ เพื่อรองรับชิ้นส่วนอื่นๆ ของคอมพิวเตอร์ ตัวเมนบอร์ดมีขนาดที่แตกต่างกันออกไปตามการใช้งาน และเมื่อมีขนาดที่แตกต่างกัน ช่องเชื่อมต่อและพอร์ตต่างๆ เองก็มีจำนวนต่างกันด้วย นั่นจึงเป็นเรื่องสำคัญที่หากใครกำลังจะประกอบคอมพิวเตอร์สักตัว การรู้จักเมนบอร์ดให้มากขึ้นจะเป็นเรื่องดีต่อการเลือกซื้อ และช่วยให้ได้สินค้าที่ตอบโจทย์การใช้งานคอมพิวเตอร์ของเรา


8.Powersupply

Power supply คือ แหล่งจ่ายไฟ หรือที่มักจะเรียกทับศัพท์ว่า เพาว์เวอร์ซัพพลาย เป็นส่วนประกอบที่สำคัญส่วนหนึ่งทำหน้าที่แปลงสัญญาณ ไฟฟ้ากระแสสลับจากแหล่งกำเนิดให้เป็นไฟฟ้ากระแสตรงด้วยความต่างศักย์ที่เหมาะสมก่อนเข้าสู่คอมพิวเตอร์ โดยมีสายเชื่อมต่อไปยังอุปกรณ์ประกอบต่าง ๆ ภายในเครื่อง ซึ่งในการแปลงสัญญาณไฟฟ้าดังกล่าวนี้จะก่อให้เกิดความร้อนขึ้นด้วย ดังนั้นภายในแหล่งจ่ายไฟจึงต้องมีพัดลมเพื่อช่วยในการระบายความร้อนออกจากตัวเครื่องซึ่งมีความสำคัญมาก เพราะการที่เครื่องมีความร้อนที่สูงมาก ๆ นั้น อาจจะเกิดความเสียหายต่ออุปกรณ์ประกอบภายในเครื่องได้ง่าย ปกติแล้วมักจะไม่ค่อยมีการเลือกซื้อพาว์เวอร์ซัพพลายกันนัก


9.HDDและSSD

อุปกรณ์คอมพิวเตอร์ที่บรรจุข้อมูลแบบไม่ลบเลือน เป็นหน่วยความจำถาวร กล่าวคือ เป็นหน่วยความจำคอมพิวเตอร์ที่สามารถเก็บรักษาข้อมูลได้โดยไม่ต้องอาศัยไฟฟ้า และปัจจุบันนั้นนิยมใช้ SSDเพราะเป็นเทคโนโลยีสมัยใหม่และยังมีประสิทธิภาพที่เหนือกว่าอย่างเห็นได้ชัด
